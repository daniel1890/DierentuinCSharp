De opdracht is om een dierentuin te bouwen, binnen deze dierentuin bevinden een aantal habitats en binnen deze habitats bevinden zich dieren.
De bedoeling is om met overerving zo min mogelijk dubbele code te hebben in het programma, je wilt dus een abstracte klasse Dier en een abstracte klasse Habitat ontwerpen
en hieruit wil je de dieren en habitats bouwen.
